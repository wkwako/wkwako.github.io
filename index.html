about

I'm a physicist-turned-computer scientist with a BA in Physics and minors in Math and Philosophy, currently finishing an MS in CS at Georgia Tech. I love puzzles: jigsaw puzzles, spatial puzzles, computational puzzles, and I'm especially drawn to problems at the intersection of education, AI, and interactive systems. Courses like HCI, Game Design, AI, and Game AI have felt less like coursework and more like permission to dig into things I'd be thinking about anyway.

More recently, I worked on a project that involved creating a web application that generates computational practice problems for students to solve, with the goal of helping students improve code comprehension skills that often go undertrained. I presented this work at NCME, a national conference on measurement in education. I look forward to continuing to apply what I've learned to areas about which I'm passionate.

Outside of academics, my hobbies include video games, D&D, playing the oboe, going on walks, and crocheting.

  • Jump King AI: Benchmarking Reinforcement and Imitation Learning on a Deterministic Platformer

    In Progress
    Game AI Machine Learning Python C# Gymnasium PPO Behavioural Cloning Stable Baselines3 dnSpy .NET / Visual Studio

    An ongoing research project comparing RL, BC, A*, and hybrid AI approaches on Jump King — a deterministic platformer where the game's predictability serves as a controlled testbed. I built a full pipeline from scratch, from real-time C# game state extraction using tools like dnSpy and Microsoft's .NET framework, to a custom Gymnasium environment with BC with per-screen data processing, PPO with BC weight transfer and value pretraining, teleporter-based curriculum training, custom reward shaping, and a class-based model management system. Write-up and images in progress.

  • Looking for the Baryon Acoustic Oscillation Signal in the 2MASS Redshift Survey using the Wavelet Transformation

    Presented @ AAS 223
    Cosmology Signal Processing Python NumPy SciPy Data Analysis

    Applied wavelet transformations to search for baryon acoustic oscillations (BAOs), which are ancient density patterns imprinted on the large-scale structure of the universe. Identified two BAO signals in the 2MASS Redshift Survey with only a 0.91% probability of occurring by chance, which corrobotes previous studies. This marks the first application of this method to the 2MRS. Produced a technical writeup to make the analysis accessible for users without a background in cosmology.

  • Ravens' Progressive Matrices Solver

    Computer Vision Knowledge-Based AI Python Rule-Based Systems Image Processing

    Built a Python program that automatically solves Ravens' Progressive Matrices, which are abstract visual reasoning puzzles used in IQ testing. The solver analyzes pixel-level relationships between matrix cells to identify the underlying visual rule and select the correct answer.
